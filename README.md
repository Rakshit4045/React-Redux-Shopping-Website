# React-Redux-Shopping-Websit

> This is a React-Redux based E-commerse site.
> This project uses api from FakeApiStore to fetch data.

### ✨ [Live Demo](https://earnest-unicorn-afcf9e.netlify.app/)

## Install

```sh
npm install
```

## Usage

```sh
npm start
```
